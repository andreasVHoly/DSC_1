README for Assignment 2 DSC
VHLAND002


To compile the script:
1. Navigate to the directory that contains the script 
2. Type 'make'

To run the script:
1. Compile
2. Navigate to the directory that contains the script 
3. Type 'make runinput'


You will be prompted to run the file in random (r) mode or manual (m) mode.
Random mode generates the content of the files randomly and manual mode lets you define the contents of the resources, files and users files.
This will create a series of files depening on what option you select with the acompanying parameters file. 
All these textfiles can be compied over and can be used with the GridSim simulator.
